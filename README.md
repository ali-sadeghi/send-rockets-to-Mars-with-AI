# send-rocket-with-AI
Send rockets to Mars with artificial intelligence(Genetic algorithm) in python.

#
## Tools

- Python 3
- EasyDraw

#
# How to Play

## Installation Requirements
```
pip install EasyDraw
```

## Run the Code

```
python main.py
```
